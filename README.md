# AldComputerService
Aplicación de escritorio en Java (Swing) para ayudar a la gestión de un taller de reparación de equipos.


## Descripción
Se trata de una aplicación simple hecha con Swing para un supuesto taller de administración y reparación de equipos con la finalidad de  practicar diferentes conceptos como la *persistencia de datos*, *patrones de diseño (Singleton)*, *la arquitectura MVC*, etc.

Este proyecto me ha permitido consolidar mis conocimientos en Java, Swing, diseño modular y buenas prácticas para un código limpio.

## Características principales
- Gestión de la información del establecimiento
- Gestión de equipos informáticos y de sus respectivas reparaciones
- Configuración de alarma
- Generación de informes


## Tecnologías utilizadas
- Java 8
- Java Swing
- JasperReports
- MariaDB
- JUnit
- NSIS


## Instalación
En el repositorio viene incluido un archivo .zip con un instalador. Lo único que hay que hacer es descomprimirlo y ejecutar el achivo .exe y seguir los pasos para generar el .JAR.
