# AldComputerService
Proyecto pequeño creada durante mi paso por el segundo año del FP de DAM en IES Antón Losada Diéguez.
Se trata de una aplicación simple hecha con Swing para un supuesto taller de administración y reparación de equipos con la finalidad de  practicar diferentes conceptos como la *persistencia de datos*, *patrones de diseño (Singleton)*, *la arquitectura MVC*, etc.
